Prefer to run it locally?

The browser sandbox is the fastest way to try Drupal CMS. But if you'd rather install it on your own machine, here's how.

For Mac / Linux

Download and run

Get Drupal CMS running locally with DDEV in a few minutes.

curl -fsSL https://drupal.org/install | bash

Full control. Set up your development environment exactly how you want it.
